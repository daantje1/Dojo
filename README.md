Aan computers vertellen wat ze moeten doen is de kern van programmeren. Wil jij dit ook kunnen? Elke donderdagavond kun je bij de DOJO leren hoe je je eigen interactieve programma's maakt, zoals een Ã©chte game! Je kunt programma's leren maken voor PC, maar uiteindelijk ook smartphone of Nintendo DS. Voorkennis is niet nodig.

Deze avond is een cursusavond, waarin samen leren programmeren voorop staat. 
Je zult aan de slag gaan met Processing, SFML of Urho3D, 
maar je bent ook vrij je eigen weg te kiezen.

 * Dag: Elke donderdagavond, behalve op feestdagen en in de kerstvakantie
 * Tijd: 
    * 18:15: inloop
    * 18:30: start les
    * 19:30: pauzed
    * 19:45: start les
    * 20:30: einde les
 * Locatie: De Jonge Onderzoekers Groningen, www.djog.nl, Dirk Huizingastraat 13
 * Kosten per leerling:
   * September t/m december: 20 euro
   * Januari tot en met mei: 20 euro
   * Juni en juli: 10 euro
 * Leeftijd: 8-25 jaar
 * Benodigdheden: Laptop met WiFi

De eerste les van het seizoen is op 8 september. Je bent ook altijd welkom om later in te stromen.
